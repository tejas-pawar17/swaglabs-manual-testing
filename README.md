# 🧪 Swag Labs Manual Testing Project

A manual testing project performed on the **Swag Labs** e-commerce web application to validate core functionalities through test scenarios, test cases, defect reporting, and execution results.

---

## 📌 Project Objective

The objective of this project is to verify the functionality and usability of the Swag Labs application by executing manual test cases, identifying defects, and documenting testing artifacts following standard software testing practices.

---

## 🌐 Application Under Test

- **Application:** Swag Labs
- **URL:** https://www.saucedemo.com/

---

## 📂 Modules Tested

| Module | Description |
|---|---|
| 🔐 Login | Verify user authentication |
| 🛍 Products | Validate product listing and navigation |
| 🛒 Add to Cart | Verify adding and removing products |
| 💳 Checkout | Validate checkout process and order placement |
| 🚪 Logout | Verify successful user logout |

---

## 🧪 Testing Types Performed

- ✔ Functional Testing
- ✔ Smoke Testing
- ✔ Regression Testing

---

## 📁 Project Structure

- 📄 README.md → Project documentation
- 📊 Test_Scenarios.xlsx → Test scenarios
- 📝 Test_Cases.xlsx → Detailed test cases
- 🐞 Bug_Report.xlsx → Defect tracking report
- 📷 Screenshots/ → Application screenshots

---

## ▶️ Test Execution Flow

1. Login using valid credentials
2. Verify Products page
3. Add product(s) to Cart
4. Complete Checkout process
5. Verify successful Logout
